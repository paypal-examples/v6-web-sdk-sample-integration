PayPal two iframe approach

The parent page is an example merchant page. Within the merchant page, there are two different iframes that each use the PayPal Web SDK. One is for the button and the second one is for the modal presentation mode container. The merchant page and iframes communicate about the state of the payment flow via postMessage.
