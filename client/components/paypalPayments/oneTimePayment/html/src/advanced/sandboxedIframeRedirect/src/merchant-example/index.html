Sandboxed Iframe Redirect Example

The parent page is an example merchant page. Within the merchant page, an iframe uses the PayPal web SDK with the redirect presentation mode. Because the iframe is sandboxed and cross-origin, it cannot navigate the top-level page on its own, so it hands the redirect URL to this merchant page via postMessage and this page performs the full-page navigation on the iframe's behalf.
