One-Time Payment - Card Fields inside a sandboxed iframe

This page embeds the Card Fields 3D Secure example inside a <iframe sandbox="...">. It demonstrates the sandbox token set a merchant must grant so that both the Card Fields components and the 3D Secure challenge modal keep working.

The sandbox tokens applied to the frame are read from the ?sandbox= query string (space-separated, URL-encoded) so the combination can be experimented with. When omitted, the recommended set below is used.
